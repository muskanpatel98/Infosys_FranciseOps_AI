# Milestone 1 - User Authentication Module

## Overview

This milestone focuses on building a secure User Authentication Module using Streamlit. The application allows users to register, log in securely using JWT authentication, recover their password using Gmail OTP verification, and access a protected dashboard after successful authentication.

## Features Implemented

- User Signup
- User Login
- JWT-based Authentication
- Forgot Password
- Gmail OTP Verification
- Password Reset
- Protected Dashboard
- Username Validation
- Password Hashing
- Security Questions for Password Recovery

## Tech Stack

- Python
- Streamlit
- JWT (JSON Web Tokens)
- Gmail SMTP
- ngrok
- Google Colab

## How to Run

1. Open the Google Colab notebook.
2. Install all required Python packages.
3. Configure Gmail App Password and Colab Secrets.
4. Start the Streamlit application.
5. Launch the application using ngrok.
6. Open the generated public URL in your browser.
